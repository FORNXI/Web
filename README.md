# Nexus AI Platform

Nexus AI is a comprehensive, multi-functional AI platform built with React, Vite, and the Google Gemini API. It provides a suite of powerful tools for developers, creators, and researchers, all within a beautiful, customizable interface.

![Nexus AI Banner](https://picsum.photos/seed/nexusai/1200/400)

## 🚀 Features

### 1. Expert AI Models
- **Code Architect:** Specialized in system design, debugging, and complex logic.
- **Scientific Researcher:** Focused on data analysis, academic rigor, and factual accuracy.
- **Strategic Analyst:** Ideal for business planning, market analysis, and decision making.
- **Google Search Integration:** Real-time grounding for up-to-date information and citations.

### 2. Image Studio
- **Advanced Generation:** Powered by Gemini 2.5 Flash Image.
- **Customizable Parameters:**
  - **Quality:** Standard, 4K, and 8K options.
  - **Aspect Ratio:** 1:1, 16:9, and 9:16 support.
  - **Styles:** Realistic, Animated, Cartoon, Cinematic, and Cyberpunk.

### 3. Intelligent Chat
- **Multiple Modes:** Fast, Deep Search, Preview (UI/UX), Creative, and Technical.
- **Real-time Interaction:** Fluid chat interface with markdown support and code highlighting.

### 4. 10+ AI Utilities
- **Summarizer:** Condense long texts into key points.
- **Translator:** Support for 10 languages (EN, ES, PT, FR, DE, IT, JA, KO, ZH, RU).
- **Grammar Checker:** Professional editing and proofreading.
- **Code Formatter:** Clean and organize your code snippets.
- **Unit Converter:** Fast transformations for various measurements.
- **Password Generator:** Create secure, cryptographically strong passwords.
- **Text-to-Speech:** Generate natural-sounding speech suggestions.
- **JSON Validator:** Verify and format data structures.
- **Regex Tester:** Test and debug regular expressions.
- **Lorem Ipsum:** Generate placeholder text for designs.

### 5. Personalization
- **10 Visual Themes:** Dark, Light, Gradient, Midnight, Aquatic, Forest, Sunset, Cyber, Minimal, and Royal.
- **Multi-language Support:** Fully localized interface in 10 major languages.

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS 4, Framer Motion
- **AI Engine:** Google Gemini API (@google/genai)
- **Icons:** Lucide React
- **Syntax Highlighting:** PrismJS

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/nexus-ai-platform.git
   cd nexus-ai-platform
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🌐 Deployment

This project is ready to be deployed on platforms like **Vercel**, **Netlify**, or **Google Cloud Run**.

### Deploy to Vercel/Netlify
1. Push your code to a GitHub repository.
2. Connect the repository to Vercel/Netlify.
3. Add `VITE_GEMINI_API_KEY` to your environment variables in the platform's dashboard.
4. The build command is `npm run build` and the output directory is `dist`.
